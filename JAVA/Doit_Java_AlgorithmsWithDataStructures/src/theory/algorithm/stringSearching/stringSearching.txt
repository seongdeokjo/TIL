문자열 검색 : 어떤 문자열 안에 다른 문자열이 들어 있는지 조사하고 들어 있다면 그 위치를 찾아내는 것을 말한다.
		  - 검색할 문자열 : 패턴(pattern), 문자열 원본 : 텍스트(text) 
1. 브루트-포스법(brute force method) : 선형 검색을 확장한 알고리즘으로 단순법, 소박법이라고도 한다.
		  - 특징 : 이미 검사를 진행한 위치를 기억하지 못하므로 효율이 좋치 못하다.	

2. KMP법 : 다른 문자를 만나면 패턴을 1칸씩 옮긴 다음 다시 패턴의 처음부터 검사하는 브루트-포스법과는 다르게 중간 검사 결과를 효율적으로 사용하는 알고리즘이다.
		  - 특징 : 검사했던 위치 결과를 버리지 않고 이를 효율적으로 활용한다.
		  		  텍스트와 패턴이 겹치는 부분을 찾아내어 검사를 다시 시작할 위치를 구한다.
		   		  패턴을 최소의 횟수로 옮겨 알고리즘의 효율을 높인다.	
		   		  하지만 몇 번째 문자부터 다시 검색을 시작할지 패턴을 이동시킬 때마다 다시 계산해야 한다면 높은 효율을 기대할 수 없다.
		   		  그래서 '몇 번째 문자부터 다시 검색하지'에 대한 값을 미리 '표'로 만들어 이 문제를 해결한다.
		   		  표를 작성할 때는 패턴 안에서 중복되는 문자의 나열을 먼저 찾아야 한다.

3. Boyer-Moore법 : 브루트-포스법을 개선한 KMP법보다 효율이 더 우수하기 때문에 실제로 문자열 검색에 널리 사용하는 알고리즘이다.
		  - 특징 : 패턴의 마지막 문자부터 앞쪽으로 검사를 진행하면서 일치하지 않는 문자가 있으면 미리 준비한 표에 따라 패턴을 옮길 크기를 정한다.
		  		  각각의 문자를 만났을 때 패턴을 옮길 크기를 저장할 표(건너뛰기 표)를 미리 만들어야 한다.
		  - 패턴에 들어 있지 않은 문자를 만난 경우
		  		- 패턴을 옮길 크기는 n이다.
		  - 패턴에 들어 있는 문자를 만난 경우
		  		- 1. 마지막에 나오는 위치의 인덱스가 k이면 패턴을 옮길 크기는 n - k - 1이다.
		  		- 2. 같은 문자가 패턴 안에 중복해서 들어 있지 않다면, 패턴을 옮길 크기는 n이다.
