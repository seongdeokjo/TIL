반복문 : 어떤 조건이 성립하는 동안 처리(프로그램 명령문 또는 명령어의 집합)를 반복하여 실행하는 것을 반복(repetition) 구조라 하며 루프(loop)라고 한다.

1. while 문 반복 -> while(제어식) 명령문
			   -> 실행 전에 반복을 계속할지를 판단하는데, 이런 구조를 '사전 판단 반복 구조'라고 한다.
		
			   1-1 do문 : do문 while(제어식);
			 		  -> 일단 루프 본문을 한 번 실행한 다음에 계속 반복할 것인지를 판단하는 '사후 판단 반복문 구조'.
* 사전 판단 반복과 사후 판단 반복의 차이점 : 사전 판단 반복문은 처음에 제어식을 평가한 결과과 0(혹은 참이 아닐 경우) 이면 루프 본문은 한 번도 실행되지 않는다.
								  이와 달리 사후 판단 반복문인 do문은 루프 본문이 반드시 한 번은 실행된다.			 		   		
		
		

2. for 문 반복 -> 하나의 변수를 사용하는 반복문은 while 문 보다 for문을 사용
			 -> 초기화 부분은 for문을 실행하기 전에 한 번만 실행, 제어식을 평가한 값이 true면 for문의 명령문을 반복. 명령문을 실행한 다음에는 업데이트 부분을 실행 
				for(1.초기화 부분; 2.제어식; 3.업데이트 부분) 명령문
			 -> 세 부분 모두 생략 가능하지만 세미콜론은 생략 불가
			 -> for문의 초기화 부분에서 선언한 변수는 for문 안에서만 유효,
			 	for문을 종료한 다음에도 변수를 사용하려면 for문 바깥 부분에서 변수를 선언해야 한다.
