논리 연산자

1. 논리곱 연산자 -> && : 둘 다 참이면 참
2. 논리합 연산자 -> || : 하나라도 참이면 참

논리 연산자의 단축 평가 : 논리 연산의 식 전체를 평가한 결과가 왼쪽 피연산자의 평가 결과만으로 정확해지는 경우
					오른쪽 피연산자의 평가를 수행하지 않는데, 이를 단축평가(short circuit evaluation)라고 한다.


드모르간 법칙 : 각 조건을 부정하고 논리곱 -> 논리합 / 논리합 -> 논리곱 으로 바꾸고 다시 전체를 부정하면 원래의 조건과 같다.
			ex) 1. x && y => !(!x || !y)
				2. x || y => !(!x && !y)
